# Rubric

The orchestrator is acceptable when it forces source-grounded routing instead of generic planning.

Required output fields:

- selected pipeline
- specialist packages
- skills
- stages
- source evidence required
- blockers
- next command or prompt

Required behavior:

- Identify source truth before choosing outputs.
- Require one typed stage manifest per major stage.
- Label facts as measured, designed, generated, inferred, or unknown.
- Stop on missing source files, stale manifests, validation contradictions, or blockers.
- Never claim final-ready status from file existence, screenshots, or summaries alone.
