# Changelog

## 0.1.0 - 2026-05-18

- Added `cumuluse` npm package with a polished black-first React agent panel, Cumulus CSS tokens, client, provider, and CLI.
- Added `npx cumuluse init` for Vite React, Next.js, and generic React projects.
- Added generated host integration files under `src/cumuluse/` without silently rewriting app source.
- Added managed `.cumuluse/venv` backend setup and package scripts for doctor, serve, and dev.
- Added `cumuluse-backend` Python package metadata and `cumuluse_backend` namespace.
- Added durable local backend foundation with SQLite WAL, uploads, ingestion tiers, approvals, diagnostics, SSE, WebSocket, Codex app-server adapter, and degraded `codex exec --json` fallback.
- Added MIT license.
